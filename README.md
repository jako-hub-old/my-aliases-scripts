Commands to simplify my daily

Instructions: 
Clone the repository in your `home`
```
cd ~
mkdir scripts
cd scripts 
git clone git@github.com:jako-hub/my-aliases-scripts.git .
cd ..
```

Edit the .bashrc
```
nano ~/.bashrc
```
Insert this line
```
source ~/scripts/.register_scripts
```
Save the file and run
```
source .bashrc
```
